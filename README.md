# Royalshetkari Information Technology

Royalshetkari Information Technology is an innovative agri-tech startup dedicated to transforming agriculture through modern technology and artificial intelligence.

## Features
- **Multi-language Support**: English, Hindi, and Marathi translations.
- **Modern UI**: Developed with React and Vite for high performance.
- **Responsive Design**: Fully mobile-friendly using Bootstrap 5.
- **Interactive Forms**: Custom forms for internships and farmer data collection.
- **Animations**: Integrated with AOS (Animate On Scroll) and Framer Motion for a premium feel.

## Tech Stack
- **Frontend**: React.js, Vite
- **Styling**: Bootstrap 5, Custom CSS
- **Animations**: Framer Motion, AOS
- **Internationalization**: i18next
- **Icons**: FontAwesome, React Icons, Bootstrap Icons

## Project Structure
- `src/components/`: Modular React components for each section.
- `src/locales/`: JSON translation files for i18n.
- `src/assets/images/`: Optimized image assets (including company logo).
- `src/styles/`: Global styles and custom CSS.
- `old_website_backup/`: Backup of the original HTML/JS static website.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
